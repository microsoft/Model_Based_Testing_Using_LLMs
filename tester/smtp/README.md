# Setting up SMTP Servers

## AioSMTPD

- Install with `python -m pip install aiosmtpd`
- Then run `python server_aiosmtpd.py` to start the server.

## SMTPD

- Comes with python 3.8 (create a virtual enviroment)
- Run `python server_smtpd.py` to start server.

## OpenSMTPD

- Install with `apt install opensmtpd`
- Then run `python server_opensmtpd.py` to start server.
